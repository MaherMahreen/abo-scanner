# ABO Scanner Pro

ABO Scanner Pro adalah scanner saham syariah Indonesia yang dibuat dengan Python.

## Fitur

- Scan seluruh saham syariah
- Sideways Detector
- Breakout Detector
- Volume Spike
- EMA Trend
- Nilai Transaksi
- ABO Score
- Ranking Top 20
- Telegram Alert
- GitHub Actions Automation

## Struktur

config.py

data.py

indikator.py

signal.py

score.py

scanner.py

telegram.py

main.py

requirements.txt

saham_syariah.csv

## Cara Kerja

1. Membaca daftar saham syariah
2. Mengambil data Yahoo Finance
3. Menghitung indikator
4. Mencari sinyal
5. Memberi ABO Score
6. Mengurutkan hasil
7. Mengirim Top 20 ke Telegram
